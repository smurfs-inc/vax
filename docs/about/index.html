About

KnowYourVaccine is created by Silvia Gheorghita and Dane Harrigan, software engineers. It's a tool to help regular people get unbiased, reliable information about the adverse effects reported in the US after vaccination with each of the Covid19 vaccines approved for emergency use by the CDC.

What is your data source?

This page uses all public data from the CDC Vaccine Adverse Event Reporting System (VAERS) VAERS accepts reports of adverse events and reactions that occur following vaccination. Healthcare providers, vaccine manufacturers, and the public can submit reports to the system. For information about the limitations of VAERS data, read the CDC's disclaimer.

How is the data presented?

This tool loads all reports from VAERS complete with symptoms and notes for each report. Symptoms reported less than 100 times are filtered out. Then, the remaining symptoms reported are divided into 10 basic categories based on severity and medical area, to make this large dataset more accessible to regular people. Reports are then displayed by age group and sex.

How often is this data updated?

The information on this page is updated once per week. The CDC releases new VAERS data weekly.

Is KnowYourVaccine only for Covid19 vaccines or other vaccines too?

Only reports for the Covid19 vaccines are represented so far.

Does KnowYourVaccine include reports from around the world?

No, only reports from the US are included so far.

Where can I see how the code works?

This project is open sourced and the code is available on GitHub.
